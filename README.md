# React Redux Counter Demo

A hands-on learning project built with **React**, **Redux Toolkit**, and **Vite**. This demo covers core React Redux concepts including global state management, slices, actions, and shared state across multiple components — without prop drilling.

## 🚀 Tech Stack

- React 19
- Redux Toolkit + React Redux
- React Router DOM
- React Bootstrap + Bootstrap 5
- Font Awesome Icons
- Vite

## 📚 What's Covered

- Setting up a Redux store with `configureStore`
- Creating slices with `createSlice` (actions + reducers in one place)
- Reading state with `useSelector`
- Dispatching actions with `useDispatch`
- Shared global state across sibling components (no props needed)
- React Router layouts, nested routes, and dynamic params
- Bootstrap-based responsive UI

## 🏃 Getting Started

```bash
npm install
npm run dev
```
